# RTC Library

## Methods

### `begin()``

Initializes the internal RTC. begin() needs to be called before any other RTC library methods.

#### Syntax

``` Arduino
rtc.begin()
```

#### Parameters

none

### `setHours()``

Set the RTC hours value.

#### Syntax

rtc.setHours(byte hours)

#### Parameters
hours: the hours value to be set.
